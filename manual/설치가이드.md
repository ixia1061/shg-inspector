## 이 프로그램은 어떤 방식인가요?

이 프로그램은 **앱스토어에서 내려받는 앱이 아니라, 인터넷 주소로 접속한 뒤 "홈 화면에 추가"하면 앱처럼 쓰는 방식**(PWA)입니다.
따라서 설치 파일(.exe, .apk)이 없고, **아래 주소에 접속만 하면 바로 사용**할 수 있습니다.

### 접속 주소 (PC·휴대폰 공통)

```
https://shg-inspector.vercel.app
```

> **참고 (팁):** 컴퓨터·휴대폰 모두 같은 주소를 사용합니다. 관리 업무(등록·통계·인쇄)는 PC, 현장 점검은 휴대폰이 편리합니다.

> **주의:** 로그인 계정은 **시스템관리자가 발급**합니다. 계정이 없으면 로그인할 수 없습니다. 계정 발급은 관리자에게 요청하세요.

## 3-1. 안드로이드(Android) 휴대폰에 설치하기

`[스크린샷: 01_설치_안드로이드_크롬메뉴.png]`
`[스크린샷: 02_설치_안드로이드_앱설치.png]`

1. **크롬(Chrome)** 앱을 엽니다.
2. 주소창에 `https://shg-inspector.vercel.app` 를 입력하고 이동합니다.
3. 화면 오른쪽 위 **점 3개(⋮) 메뉴**를 누릅니다.
4. 목록에서 **[앱 설치]** 또는 **[홈 화면에 추가]** 를 누릅니다.
5. **[설치]** 를 누릅니다.
6. 홈 화면에 **소화기점검** 아이콘이 생깁니다. 이제 이 아이콘으로 실행하세요.

`[스크린샷: 06_홈화면_아이콘.png]`

> **팁:** 홈 화면 아이콘으로 실행하면 주소창이 없는 전체 화면으로 열려 앱처럼 편합니다.

---

## 3-2. 아이폰(iPhone·iPad)에 설치하기

`[스크린샷: 03_설치_아이폰_공유버튼.png]`
`[스크린샷: 04_설치_아이폰_홈화면추가.png]`

1. **사파리(Safari)** 앱을 엽니다. (크롬이 아닌 사파리를 사용하세요)
2. 주소창에 `https://shg-inspector.vercel.app` 를 입력하고 이동합니다.
3. 화면 아래쪽 가운데 **공유 버튼(□에 ↑ 화살표)** 을 누릅니다.
4. 메뉴를 아래로 내려 **[홈 화면에 추가]** 를 누릅니다.
5. 오른쪽 위 **[추가]** 를 누릅니다.
6. 홈 화면에 **소화기점검** 아이콘이 생깁니다.

> **주의:** 아이폰은 **반드시 사파리**에서 "홈 화면에 추가"를 해야 카메라(QR 스캔)와 오프라인 기능이 정상 동작합니다.

---

## 3-3. 윈도우(Windows) 컴퓨터에 설치하기

`[스크린샷: 05_설치_PC_주소창설치아이콘.png]`

1. **크롬(Chrome)** 또는 **엣지(Edge)** 브라우저를 엽니다.
2. 주소창에 `https://shg-inspector.vercel.app` 를 입력하고 이동합니다.
3. 주소창 **오른쪽 끝의 설치 아이콘(모니터 모양 ⊞)** 을 누릅니다.
   - 아이콘이 안 보이면, 오른쪽 위 **점 3개 메뉴 → [앱] → [이 사이트를 앱으로 설치]** 를 누릅니다.
4. **[설치]** 를 누릅니다.
5. 바탕화면/시작 메뉴에 **소화기 점검 관리** 아이콘이 생깁니다.

> **팁:** 설치하지 않고 그냥 브라우저에서 주소로 접속해서 사용해도 모든 기능이 동일하게 동작합니다. 설치는 "바로가기"를 만드는 것에 가깝습니다.

## 3-4. 처음 로그인하기

`[스크린샷: 10_로그인화면.png]`
`[스크린샷: 11_로그인_이메일비밀번호입력.png]`

1. 설치한 **소화기점검** 아이콘(또는 주소)으로 접속합니다.
2. **이메일** 칸에 발급받은 이메일을 입력합니다.
3. **비밀번호** 칸에 발급받은 비밀번호를 입력합니다.
4. **[로그인]** 버튼을 누릅니다.
5. 역할에 따라 화면이 자동으로 나뉩니다.
   - **관리자** → 대시보드 화면
   - **점검자** → QR 스캔 화면

> **주의:** 공개 회원가입은 없습니다. 로그인이 안 되면 이메일/비밀번호를 확인하고, 그래도 안 되면 시스템관리자에게 계정 상태(활성 여부·비밀번호 재설정)를 문의하세요. (자세한 해결은 `FAQ.md` 참고)

## 설치가 잘 되었는지 확인하는 체크리스트

- [ ] 홈 화면(또는 바탕화면)에 **소화기점검** 아이콘이 생겼다.
- [ ] 아이콘을 누르면 로그인 화면이 뜬다.
- [ ] 발급받은 계정으로 로그인이 된다.
- [ ] (점검자) 로그인하면 **QR 스캔** 화면이 나온다.
- [ ] (관리자) 로그인하면 **대시보드** 화면이 나온다.

문제가 있으면 `FAQ.md` 의 "자주 발생하는 오류"를 확인하세요.
