## 2. 시스템 소개

### 프로그램 목적
소화기를 **QR 코드로 등록·점검·관리**하는 프로그램입니다.
관리자는 **미점검 소화기와 내용연수(교체 시기)를 한눈에 파악**하고, 점검 기록·사진·관리대장을 관리합니다.

### 주요 기능 (관리자)
- 사업장·건물·층·차량 등 **조직 구조 관리**
- **소화기 등록** 및 QR 라벨 인쇄
- **점검 현황**(미점검/점검완료) 및 **관리대장(엑셀) 다운로드**
- **내용연수 관리**(교체 예정/만료)
- **점검 사진 관리**, **통계**
- (시스템관리자) **사용자 계정 발급·역할·담당 사업장 배정**

### 사용 대상
- **시스템관리자(super_admin)**: 전체 권한. 사업장 등록·사용자 관리 독점
- **관리자(admin)**: 배정된 담당 사업장 범위 안에서만 관리
- **점검자(inspector)**: 현장 점검(사용자매뉴얼 참고)

## 3. 로그인과 화면 구성

`[스크린샷: 10_로그인화면.png]`

1. **소화기 점검 관리** 아이콘(또는 주소)으로 접속합니다.
2. **이메일 / 비밀번호** 를 입력하고 **[로그인]** 을 누릅니다.
3. 관리자는 로그인하면 **대시보드** 화면이 나옵니다.

### 왼쪽 메뉴(사이드바) 구성

`[스크린샷: 40_관리자_대시보드.png]`

컴퓨터에서는 왼쪽에 메뉴가 고정으로 보이고, 휴대폰에서는 **왼쪽 위 햄버거(≡) 버튼**을 누르면 메뉴가 나옵니다.

| 메뉴 | 하는 일 | 누가 사용 |
|---|---|---|
| **대시보드** | 이번 달 점검/미점검·교체예정·만료·최근 이상 요약 | 관리자·시스템관리자 |
| **사업장/건물 관리** | 사업장·건물·층·차량 등록·수정 (사업장 추가는 시스템관리자만) | 관리자·시스템관리자 |
| **소화기 관리** | 소화기 등록·수정·삭제·검색 | 관리자·시스템관리자 |
| **수량 현황** | 건물×종류별 소화기 수량 집계 | 관리자·시스템관리자 |
| **점검현황** | 이번 달 미점검/점검완료, 점검률, 관리대장 다운로드 | 관리자·시스템관리자 |
| **내용연수 관리** | 교체 예정·만료 소화기 목록 | 관리자·시스템관리자 |
| **사진 관리** | 점검 사진 조회·삭제·ZIP 다운로드 | 관리자·시스템관리자 |
| **통계** | 건물별 점검률·점검자별 실적 | 관리자·시스템관리자 |
| **QR Code 관리** | QR 라벨 검색·선택·인쇄 | 관리자·시스템관리자 |
| **사용자 관리** | 계정 발급·역할·담당 사업장 배정 | **시스템관리자만** |

> **주의:** **사업장 추가/수정/삭제**와 **사용자 관리**는 **시스템관리자만** 할 수 있습니다. 일반 관리자는 자신에게 배정된 사업장의 건물·소화기·점검만 관리합니다.

## 4. 메뉴별 사용법

### 4-1. 대시보드

`[스크린샷: 40_관리자_대시보드.png]`

- 상단 카드: **전체 소화기 / 이번 달 점검 / 이번 달 미점검 / 교체 예정 / 만료 / 최근 이상**
- 아래: **건물별 이번 달 점검률**(막대 그래프, 가나다순)
- 점검/미점검 기준은 **매월 1일(한국 시간)에 초기화**됩니다.

### 4-2. 사업장/건물 관리

`[스크린샷: 41_관리자_사업장건물관리.png]`

사업장 목록에서 사업장을 누르면 그 사업장의 **건물 → 층 → 차량** 구조를 관리합니다.

**건물 추가**
`[스크린샷: 42_관리자_건물추가.png]`
1. 사업장 상세에서 **[건물 추가]** 를 누릅니다.
2. 건물 번호·이름을 입력하고 저장합니다.

**층 추가/순서 변경**
`[스크린샷: 43_관리자_층추가.png]`
1. 건물 오른쪽 **[층 추가]** 로 층을 만듭니다. (층 코드 예: 1, 2, 0=지하, R=옥상)
2. 층을 클릭해 선택한 뒤 위쪽 **▲▼ 버튼**으로 순서를 바꿉니다.

**차량 추가**
`[스크린샷: 44_관리자_차량추가.png]`
1. 건물 오른쪽 **[차량 추가]** 를 누릅니다.
2. 차량 번호·번호판·차종·관리부서를 입력합니다. (차량도 건물에 소속됩니다)

> **주의:** 사업장 자체의 추가/수정/삭제 버튼은 **시스템관리자**에게만 보입니다.

### 4-3. 소화기 관리

`[스크린샷: 45_관리자_소화기관리목록.png]`

소화기 목록을 보고 검색·등록·수정·삭제합니다.

**검색·필터**
`[스크린샷: 46_관리자_소화기검색.png]`
- **검색창**: **관리번호·제조번호·위치**(건물/층/설치위치, 차량은 번호판/차종/부서)를 한글로 검색
- **사업장** 선택, **상태**(정상/교체 90일 전/30일 전/만료/내용연수 없음) 선택
- 목록은 50개씩 페이지로 나뉩니다.

**소화기 등록** — 자세한 방법은 아래 **[5. 소화기 등록]** 참고.

**소화기 상세·수정·삭제**
`[스크린샷: 48_관리자_소화기상세.png]`
- 목록에서 **관리번호**를 누르면 상세 화면이 열립니다.
- 왼쪽에서 **정보 수정**, 오른쪽에서 **점검 이력**(사진 포함)을 봅니다.
- 상단 버튼: **[점검]**(관리자가 직접 점검), **[QR/라벨]**(라벨 인쇄), **[삭제]**

**소화기 삭제(폐기·철수)**
`[스크린샷: 49_관리자_소화기삭제.png]`
1. 상세 화면 상단 **[삭제]** 버튼을 누릅니다.
2. 확인 창에서 **그 소화기의 관리번호를 정확히 입력**해야 삭제 버튼이 활성화됩니다.
3. **[완전 삭제]** 를 누릅니다.

> **주의:** 삭제는 **되돌릴 수 없습니다.** 점검 이력이 있는 소화기는 **점검 기록과 사진까지 함께 영구 삭제**됩니다. 실제로 폐기·철수한 소화기에만 사용하세요.

> **팁:** 관리자는 QR 없이도 상세/목록의 **[점검]** 버튼으로 직접 점검을 기록할 수 있습니다. (점검자에게도 완료로 반영됩니다)

### 4-4. 수량 현황

`[스크린샷: 50_관리자_수량현황.png]`
- 상단 **사업장 버튼**을 누르면 그 사업장의 **건물 × 소화기 종류** 교차표와 요약이 나옵니다.

### 4-5. 점검현황

`[스크린샷: 51_관리자_점검현황_사업장토글.png]`
`[스크린샷: 52_관리자_점검현황_탭.png]`

1. 상단 **사업장 버튼**을 누르면 그 사업장 기준으로 화면 전체가 바뀝니다.
2. **건물별 이번 달 점검률**(막대 그래프)을 봅니다.
3. 탭으로 목록을 봅니다.
   - **이번달 미점검**: 이번 달에 아직 점검 안 된 소화기
   - **점검완료**: 이번 달에 점검된 소화기
4. 오른쪽 위 **[○○○ 관리대장]** 버튼으로 **엑셀(.xlsx) 관리대장**을 내려받습니다. (자세히는 아래 [6. 점검 결과 확인] 참고)

### 4-6. 내용연수 관리

`[스크린샷: 54_관리자_내용연수관리.png]`
- 교체 시기가 다가오거나 지난 소화기를 **교체 예정일 순**으로 봅니다.
- 상태: **교체 90일 전 / 30일 전 / 만료**. (이산화탄소·할론 등 내용연수가 없는 종류는 계산에서 제외)

### 4-7. 사진 관리

`[스크린샷: 55_관리자_사진관리.png]`
- 점검 사진을 **관리번호별로** 모아 봅니다.
- 사진을 **삭제**하거나, **ZIP 파일로 한 번에 다운로드**(관리번호별 폴더)할 수 있습니다.

### 4-8. 통계

`[스크린샷: 56_관리자_통계.png]`
- **건물별 이번 달 점검률**
- **이번 달 점검자별 실적**(누가 몇 건 점검했는지)과 **이상 점검 비율**

### 4-9. QR Code 관리 (라벨 인쇄)

`[스크린샷: 57_관리자_QR코드관리.png]`
`[스크린샷: 58_관리자_라벨인쇄.png]`

1. 소화기를 **검색·필터**한 뒤 **체크박스로 여러 개 선택**합니다.
2. **라벨 크기**(예: 50×30mm)와 위치 표시 여부를 고릅니다.
3. **인쇄** 버튼을 누르면 **한 장에 하나씩** 라벨이 인쇄됩니다.
4. QR은 관리번호로 즉시 생성되므로, 새로 등록/수정한 소화기는 **[새로고침]** 후 인쇄하세요.

> **팁:** QR은 재발급하지 않습니다. 위치 이동으로 관리번호가 바뀌어도 **옛 QR도 계속 최신 소화기로 연결**되니, 라벨을 다시 붙일 필요는 없습니다.

## 5. 소화기 등록 (새 소화기 추가)

`[스크린샷: 47_관리자_소화기등록폼.png]`

**소화기 관리** 화면에서 등록 버튼을 눌러 아래 순서로 입력합니다.

1. **위치 유형** 선택: **건물** 또는 **차량**
2. **사업장** 선택
3. **건물** 선택
4. (건물일 때) **층** 선택 / (차량일 때) **차량** 선택
5. **설치 위치** 입력 (예: `출입구 옆 소화전함`)
6. **관리번호 끝자리 (선택)**: 비워두면 **자동으로 번호가 부여**됩니다. 특정 번호로 넣고 싶을 때만 입력.
7. **소화기 종류** 선택 (분말/이산화탄소/할론 등)
8. **소화기 용량** 입력 (예: `3.3kg`)
9. **제조년월 (연.월)** 입력: **연·월만** 입력합니다. (예: `2026-12`, 숫자만 쳐도 자동 형식)
10. **제조번호** 입력 (소화기 명판의 제조번호)
11. **내용연수(년)** 입력 (분말은 보통 10년, 없는 종류는 "내용연수 없음" 선택)
12. **[등록]** 을 눌러 완료합니다.

> **주의:** **관리번호(전체)는 자동으로 만들어집니다.** 형식은 건물 소화기 `{기관}-{건물}-{층}-{번호}`(예: 공사-1-1-1), 차량 소화기 `{기관}-{건물}-차-{번호}`(예: 공사-1-차-1) 입니다.

> **주의:** 제조일은 **연·월만** 입력하며, 저장 시 자동으로 **그 달 1일**로 저장됩니다. (교체예정일 계산의 기준)

> **팁:** 관리번호 끝자리가 중복되면 저장이 막힙니다. 그럴 때는 끝자리를 **비워서 자동 부여**되게 하세요.

## 6. 점검 결과 확인 · 관리대장(엑셀) 내보내기

### 6-1. 점검 이력 조회

`[스크린샷: 48_관리자_소화기상세.png]`
- **소화기 관리 → 관리번호 클릭 → 상세 화면 오른쪽**에 그 소화기의 점검 이력(날짜·결과·비고·사진)이 시간순으로 나옵니다.
- **점검현황** 화면에서는 사업장·건물별로 미점검/점검완료를 모아 봅니다.

### 6-2. 검색·필터
- **소화기 관리**의 검색창(관리번호·제조번호·위치)과 사업장·상태 필터로 원하는 소화기를 빠르게 찾습니다.

### 6-3. 관리대장(엑셀) 다운로드

`[스크린샷: 53_관리자_관리대장다운로드.png]`

1. **점검현황** 화면에서 대상 **사업장 버튼**을 먼저 누릅니다.
2. 오른쪽 위 **[○○○ 관리대장]** 버튼을 누릅니다.
3. 그 사업장의 소화기 관리대장 **엑셀 파일(.xlsx)** 이 다운로드됩니다.
   - 파일에는 **표지(점검일자·점검자 수기란 + 동·층별 보유현황표)** 와 **점검대장(소화기별 위치·종류·제조일·최근점검일·점검결과·불량항목·조치내용 등)** 이 들어 있습니다.

> **팁:** 무안국제공항을 선택하면 무안 관리대장만, 상주업체를 선택하면 상주 관리대장만 받아집니다. 사업장별로 따로 관리하세요.

## 7. 관리자 기능 (시스템관리자 전용)

### 7-1. 사용자 관리 (계정 발급)

`[스크린샷: 59_관리자_사용자관리.png]`
`[스크린샷: 60_관리자_사용자추가.png]`

> **주의:** 이 메뉴는 **시스템관리자에게만** 보입니다.

1. **사용자 관리** 메뉴로 들어갑니다.
2. **사용자 추가** 로 이름·이메일·비밀번호·역할을 입력해 계정을 만듭니다.
3. 역할을 정합니다.
   - **관리자(admin)**: 배정된 사업장 관리
   - **점검자(inspector)**: 현장 점검

### 7-2. 권한(역할)·담당 사업장 관리

`[스크린샷: 61_관리자_담당사업장배정.png]`

1. 사용자 목록에서 역할을 바꾸거나, **담당 사업장 배정** 창에서 그 사용자가 관리·점검할 **사업장을 지정**합니다.
2. 배정된 사업장의 데이터만 그 사용자에게 보입니다.

> **주의:** 시스템관리자 계정은 보호되어 있어 **삭제·강등·비활성화가 되지 않습니다.** 일반 관리자는 다른 사람의 역할을 바꿀 수 없습니다.

### 7-3. 비밀번호 변경

`[스크린샷: 62_계정_비밀번호변경.png]`
- 본인 비밀번호는 **내 계정** 메뉴에서 바꿉니다.
- 다른 사용자가 비밀번호를 잊었을 때는 시스템관리자가 재설정을 도와줍니다.

### 7-4. 백업 / 데이터 복원 안내 (중요)

> **주의:** 현재 프로그램 **화면 안에는 "백업"·"복원" 버튼이 없습니다.** 데이터는 클라우드 데이터베이스(Supabase)에 저장되며, 그곳에서 **자동 백업**됩니다.
>
> - **일상 백업**: 별도 조작이 필요 없습니다(클라우드 자동 백업).
> - **내보내기(수동 보관)**: 필요 시 **점검현황 → 관리대장(엑셀) 다운로드**, **사진 관리 → ZIP 다운로드**로 데이터를 파일로 보관할 수 있습니다.
> - **복원**: 데이터 복구가 필요한 경우(실수 삭제 등)에는 시스템 담당자/개발자에게 요청해 데이터베이스 백업 시점으로 복구합니다. (화면에서 직접 복원하는 기능은 없습니다)

이 부분(화면 내 백업/복원)은 **아직 없는 기능**으로, 문서 끝 "부족한 기능" 목록에도 포함되어 있습니다.

## 8. 관리자용 요약 (한 장)

- 조직 구성: **사업장/건물 관리**에서 건물·층·차량 등록 (사업장은 시스템관리자)
- 소화기 등록: **소화기 관리 → 등록** (제조일은 연·월만, 관리번호 자동)
- 라벨: **QR Code 관리**에서 선택 → 인쇄
- 점검 확인: **점검현황**(미점검/점검완료) + **관리대장 엑셀** 다운로드
- 교체 관리: **내용연수 관리**
- 계정: **사용자 관리**(시스템관리자) — 역할·담당 사업장 배정

문제가 생기면 `FAQ.md` 를 확인하세요.
