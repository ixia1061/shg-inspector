## A. 설치 · 로그인

- [ ] 안드로이드에서 "홈 화면에 추가"로 설치되고 아이콘이 생긴다.
- [ ] 아이폰(사파리)에서 "홈 화면에 추가"로 설치된다.
- [ ] PC(크롬/엣지)에서 접속되고, 설치 아이콘으로 설치된다.
- [ ] 올바른 이메일/비밀번호로 **로그인**이 된다.
- [ ] 잘못된 비밀번호는 **로그인 실패 메시지**가 뜬다.
- [ ] 관리자 로그인 → **대시보드**로 이동한다.
- [ ] 점검자 로그인 → **QR 스캔** 화면으로 이동한다.

## B. 점검자 — QR 점검

- [ ] **[카메라 시작]** 을 누르면 카메라가 켜진다.
- [ ] 카메라 권한 요청이 뜨고 **허용**하면 스캔이 시작된다.
- [ ] 소화기 QR을 비추면 **점검 화면으로 자동 이동**한다.
- [ ] **[QR이 손상됐나요? 관리번호 직접 입력]** → 관리번호 입력 → **[점검 시작]** 으로 점검 화면에 들어간다.
- [ ] 관리번호를 틀리게 입력하면 "소화기 정보를 찾을 수 없습니다" 가 뜬다.
- [ ] 점검 화면에 관리번호·위치·종류·제조일·제조번호·최근점검이 맞게 나온다.
- [ ] 점검 항목 4개가 **처음에 모두 정상(체크)** 되어 있다.
- [ ] 항목 체크를 해제하고 저장하면 결과가 **"이상"** 으로 저장된다.
- [ ] 모두 정상으로 저장하면 **"정상"** 으로 저장된다.
- [ ] 비고를 입력해 저장하면 상세/관리대장에 반영된다.
- [ ] 카메라 버튼으로 사진을 찍으면 **관리번호 워터마크**가 새겨진다.
- [ ] 사진을 5장까지 추가할 수 있고, 6장째는 막힌다.
- [ ] 사진 ✕로 삭제가 된다.
- [ ] **[휴대폰에 저장]** 으로 사진이 갤러리에 저장된다.
- [ ] **[점검완료]** 후 완료 화면이 뜬다.
- [ ] **[다음 소화기 스캔]** 으로 스캔 화면으로 돌아간다.
- [ ] **[점검 현황 보기]** → 건물별 현황이 보인다.

## C. 점검자 — 오프라인/동기화

- [ ] 인터넷을 끈 상태에서 오프라인 안내 띠가 뜬다.
- [ ] 오프라인에서 점검을 저장할 수 있다.
- [ ] 인터넷을 다시 켜면 **자동 동기화**되어 서버에 반영된다.
- [ ] 동기화 완료 후 관리자 화면에 그 점검이 나타난다.

## D. 관리자 — 조직/소화기

- [ ] 대시보드 요약 카드(전체/이번달 점검·미점검/교체예정/만료/최근이상)가 나온다.
- [ ] (시스템관리자) 사업장 추가/수정이 된다.
- [ ] 건물·층·차량을 추가할 수 있다.
- [ ] 층 순서를 ▲▼로 바꿀 수 있다.
- [ ] **소화기 등록**: 위치→종류→용량→제조년월(연·월)→제조번호→내용연수 입력 후 등록된다.
- [ ] 제조년월을 `2026-12`처럼 입력하면 저장되고, **그 달 1일**로 저장된다.
- [ ] 관리번호 끝자리를 비우면 **자동 부여**된다.
- [ ] 관리번호 끝자리를 중복 지정하면 저장이 막힌다.

## E. 관리자 — 검색/현황/대장

- [ ] 소화기 검색이 **관리번호**로 된다.
- [ ] 소화기 검색이 **제조번호**로 된다.
- [ ] 소화기 검색이 **위치(한글: 건물/층/설치위치)** 로 된다.
- [ ] 사업장/상태 필터가 동작한다.
- [ ] 점검현황에서 **사업장 버튼**을 누르면 그 사업장만 표시된다.
- [ ] **이번달 미점검 / 점검완료** 탭이 올바르게 나뉜다.
- [ ] **관리대장(엑셀)** 이 선택 사업장 기준으로 다운로드된다.
- [ ] 관리대장 표지(점검일자·점검자란, 보유현황표)와 점검대장이 채워져 있다.
- [ ] 관리대장 최근점검일이 **한국 시간 기준**으로 맞다.

## F. 관리자 — 기타 메뉴

- [ ] 수량 현황이 사업장별 교차표로 나온다.
- [ ] 내용연수 관리에 교체예정/만료가 나온다.
- [ ] 사진 관리에서 사진 조회·삭제·ZIP 다운로드가 된다.
- [ ] 통계에 건물별 점검률·점검자별 실적이 나온다.
- [ ] QR Code 관리에서 다중 선택 후 라벨이 한 장씩 인쇄된다.

## G. 관리자 — 삭제/권한

- [ ] 소화기 상세에서 **관리번호 입력** 후 삭제가 된다.
- [ ] 삭제 시 그 소화기의 점검 이력·사진도 함께 사라진다.
- [ ] 일반 관리자에게는 **사용자 관리 메뉴가 안 보인다.**
- [ ] (시스템관리자) 사용자 추가·역할 변경·담당 사업장 배정이 된다.
- [ ] 배정된 사업장의 데이터만 해당 관리자/점검자에게 보인다.
- [ ] 내 계정에서 비밀번호를 바꿀 수 있다.

## H. 날짜/시간 (한국 시간)

- [ ] 점검/미점검 기준이 **매월 1일(한국 시간)** 에 초기화된다.
- [ ] 자정~오전 사이에 점검해도 날짜가 하루 어긋나지 않는다.
